The Overview

Angular

  • Open-sourced JavaScript framework
  • Launched in September 2016
  • Managed by Google
  • For web and mobile
  • TypeScript-based

React

  • JavaScript library for UI development
  • Introduced in May 2013
  • Managed by Facebook
  • For web and mobile
  • Virtual DOM

Usage

Similarities

  • Open source
  • Component-based architecture
  • Client-side and server-side rendering
  • Performance
  • Easy to update

Learning Curve

Angular

  • TypeScript and Workflow
  • Components, modules and syntax
  • Dependency Injection
  • Optimizations

React

  • ES6 and JSX
  • "Everything is JavaScript"
  • State Management
  • Optimizations
